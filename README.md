# Internship Task 03: Minimalist Tic-Tac-Toe Web Application

A clean, modern, and user-friendly Tic-Tac-Toe web application built as part of my web development internship assignment. The project features a neo-minimalist light user interface focused on high readability, seamless animations, and clean state tracking.

## 🚀 Live Demo
Check out the live deployment here: [sct-wd-03-nine.vercel.app](https://sct-wd-03-nine.vercel.app/)

## ✨ Key Features
- **Dynamic Setup Mode:** Prompts for custom player names based on selected mode.
- **Smart Game Engine:** Supports smooth local Player vs Player and a responsive Computer AI mode.
- **Vibrant Match Celebrations:** Built-in automatic performance canvas confetti triggers on human victories.
- **Custom Game Over States:** Shows engaging win screens and custom match-lost layouts (😢 Better luck next time!) when playing the computer.
- **Premium UI Arena:** Built using a high-contrast Neo-Minimalist Light design for superior usability.

## 🛠️ Tech Stack
- **HTML5** (Semantic Structure)
- **CSS3** (Flexbox, Grid Layouts, Custom Variables)
- **JavaScript (ES6+)** (State Tracking Engine, DOM Manipulation)
- **Canvas Confetti Library** (High-Performance Particle Rendering)
